# Olive Oil Regional Origin Analysis
## STA 9705 — Group Project (Baruch College, Spring 2026)

### Project Overview
This project investigates whether the chemical composition of olive oil can be used to distinguish which region of Italy it originated from. Using a dataset of 572 Italian olive oil samples, we apply five multivariate statistical techniques in SAS to explore, model, and classify regional origin based on eight fatty acid measurements.

### Research Question
**Can the chemical composition of olive oil be used to distinguish which region it originated from?**

### Dataset
* **Source**: Olive Oil dataset from the tourr package in R
* **Observations**: 572 olive oil samples
* **Variables**: 10 (region, area, and 8 fatty acids: palmitic, palmitoleic, stearic, oleic, linoleic, linolenic, arachidic, eicosenoic)
